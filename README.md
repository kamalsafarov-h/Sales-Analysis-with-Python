# Sales Performance Analysis

Python-based data analysis project using Sales Performance dataset.

## What this project covers

- Data cleaning and transformation (price formatting, full name creation)
- Merging multiple Excel sheets into one dataframe
- Regional and category-based sales analysis
- Statistical analysis (Z-score, IQR, T-test, Pearson correlation)
- Data visualization (Bar, Scatter, Pie, Line, KDE, Regplot charts)

## Technologies used

- Python 3.x
- Pandas
- Matplotlib
- Seaborn
- Scipy


## Analysis included

- Top revenue by worker
- Sales by region
- Category market share
- Monthly sales trend
- Outlier detection
- Hypothesis testing (Baki vs Quba regions)
- Z-score distribution
